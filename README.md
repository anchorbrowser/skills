# AnchorBrowser Skills

Install the AnchorBrowser CLI skill with the Vercel Skills CLI.

## Install

```bash
# Install this skill to all supported agents (recommended)
npx skills add anchorbrowser/skills --skill anchorbrowser-cli --all
```

```bash
# Or install all skills in this repo to all supported agents
npx skills add anchorbrowser/skills --all
```

## List Available Skills

```bash
npx skills add anchorbrowser/skills --list
```
